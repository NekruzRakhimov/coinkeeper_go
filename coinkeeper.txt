Техническое задание для консольного приложения для учета финансов на Golang:

Название проекта: Finance Tracker

Описание:
Консольное приложение для отслеживания личных финансов.
Приложение должно позволять пользователю выполнять следующие действия:
Добавление новой финансовой операции (доход или расход).
Просмотр всех существующих финансовых операций.
Редактирование существующей финансовой операции.
Удаление финансовой операции.
Отображение итоговой суммы доходов и расходов.
Фильтрация финансовых операций по дате, категории или сумме.

Функциональные требования:
Пользователь может добавлять новые финансовые операции.
Каждая операция должна иметь следующие поля:
Уникальный идентификатор
Тип операции (доход или расход)
Категория (например, "Зарплата", "Продукты", "Транспорт")
Сумма
Дата операции
Описание (необязательно)

Пользователь может просматривать список всех существующих финансовых операций.
Пользователь может редактировать существующую финансовую операцию. Он может изменить тип, категорию, сумму, дату или описание.
Пользователь может удалять финансовую операцию из списка.
Приложение должно отображать итоговые суммы доходов и расходов.
Пользователь может фильтровать финансовые операции по дате, категории или сумме.

Технические требования:
Язык программирования: Golang
Форматирование вывода: использование стандартного пакета fmt

Дополнительные функции (необязательные):
Создание бюджетов и отслеживание расходов в рамках бюджета.
